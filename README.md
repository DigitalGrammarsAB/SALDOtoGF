# SALDO to GF

Script for importing SALDO as a GF grammar
Based primarily on the work of Malin Ahlberg: <https://github.com/MalinAhlberg/SwedishProject/tree/master/saldo>

## Usage

```
runghc Main.hs data/saldo.json
```
